# Diwali_Sales-_Analysis_Python
Performed Exploratory Data Analysis (EDA) on Diwali Sales data using Python to uncover customer behavior insights and improve the overall customer experience.

 Bullet-Point Summary for EDA on Diwali Sales Data:

Analyzed Diwali sales dataset to identify patterns in customer purchases and product preferences.

Performed data cleaning (handling missing values, correcting data types, etc.) using pandas and numpy.

Visualized key insights using matplotlib and seaborn (e.g., top-selling product categories, purchase behavior by gender and age group).

Segmented customers by demographics such as age, gender, marital status, and occupation.

Identified trends such as:

Which age group spends the most

Which product categories are most popular during Diwali

Most buyers are from which category
